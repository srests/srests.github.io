【Error】Tomcat:ZipException

2018-06-08

maven打包之后，tomcat启动时报错：
java.util.zip.ZipException: error in opening zip file
原因为其中使用的jar包不正确，对依赖jar包逐个排查，找到异常jar重新替换为正常jar包即可。
